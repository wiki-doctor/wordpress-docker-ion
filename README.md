# wordpress-docker
wordpress docker compose with ioncube and sourceguardian

## who to conifg 
-- just open ```.env``` and set your environment

## who to start without PhpMyAdmin 
-- just run this command :
```docker compose up -d --build wp-web ```
## who to full start 
-- run this command :
``` docker compose  up -d ```
